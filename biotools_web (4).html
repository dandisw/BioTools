BioTools DSW
Platform bioinformatika berbasis web menggunakan Claude AI sebagai mesin analisis. Semua fitur dari program lokal (BLAST, Phylo, Primer, Plasmid) kini dapat diakses dari browser mana saja tanpa instalasi.
BioPython compatible NCBI GenBank Claude Sonnet 4 Gratis & Open Universitas Jenderal Soedirman
🔬
BLAST Identifier
Identifikasi sekuens ke GenBank NCBI. Hitung % identity, E-value, query coverage. Filter diversitas spesies.
🌿
Pohon Filogenetik
Bangun pohon NJ/UPGMA dari sekuens FASTA. Analisis matriks jarak dan interpretasi hubungan evolusi.
🧬
Desain Primer
Desain primer PCR & sequencing. Kalkulasi Tm SantaLucia, GC%, self-complementarity, hairpin.
⭕
Desain Plasmid
Peta plasmid lingkaran, in silico kloning, simulasi restriction digest, anotasi fitur.
🔄
Konversi Sekuens
Reverse complement, translasi, GC content, validasi, statistik sekuens.
📖
Cara Penggunaan
1. Masukkan API Key Anthropic. 2. Pilih fitur. 3. Input sekuens. 4. Klik jalankan.
⚠ Diperlukan API Key Anthropic (Claude) untuk menggunakan semua fitur. Dapatkan di console.anthropic.com. API Key tidak disimpan di server — hanya ada di browser Anda.
